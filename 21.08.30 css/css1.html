CSS 스타일 맛 보기

CSS는 Cascading Style Sheet 의 약자로
HTML같은 문서의 스타일을 꾸밀 때 사용하는 스타일 시트 언어입니다
글꼴이나 배경색, 너비와 높이, 위치 등을 지정합니다
(태그)selector(h3){속성(color):속성값("#00ff00") ;}

HTML 문서에 CSS 스타일 시트 만드는 방법 3가지

  1. 태그에 스타일 시트 작성
    selecteor{속성:속성값;}
  2. style 속성에 스타일 시트 작성
    style=""
  3. CSS 파일을 별도로 만들어 html 문서에 연결
    , @import로 불러 사용
